# 📝 Python To-Do List (Backend Basics Project)

Ein einfaches To-Do-Listen-Projekt in Python, das Aufgaben in einer JSON-Datei speichert.
Perfekt, um Grundlagen wie Dateiverarbeitung, JSON und einfache CLI-Logik zu lernen.

---

## 🚀 Features

* Aufgaben hinzufügen
* Aufgaben anzeigen
* Aufgaben löschen
* Speicherung in `tasks.json`
* Einfache Kommandozeilen-Bedienung

---

## 📁 Projektstruktur

```
backend-roadmap/
│
├── Python-basics-project.py   # Hauptprogramm
├── tasks.json                # Speicher für Aufgaben
└── README.md                 # Projektdokumentation
```

---

## ⚙️ Voraussetzungen

* Python 3.x installiert

Download: https://www.python.org/

---

## ▶️ Projekt starten

Im Terminal:

```bash
python Python-basics-project.py
```

oder (wie bei dir):

```bash
C:/Users/.../python.exe Python-basics-project.py
```

## 🧠 Wichtiger Hinweis

Falls folgender Fehler auftritt:

```
json.decoder.JSONDecodeError
```

👉 Stelle sicher, dass deine `tasks.json` Datei **nicht leer** ist.

Minimaler Inhalt:

```json
[]
```

---

## 💡 Lernziele

Dieses Projekt hilft dir, folgende Python-Konzepte zu verstehen:

* Arbeiten mit Dateien (`open`, `read`, `write`)
* JSON verarbeiten (`json.load`, `json.dump`)
* Funktionen und Programmstruktur
* Fehlerbehandlung (`try/except`)
